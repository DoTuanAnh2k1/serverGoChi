Command Catalog

Import a pool of raw command patterns (XML or CSV), then bulk-apply NE profile / category / risk level to many of them at once. The pool is stored in your browser (localStorage); re-import any time. The "Bulk Apply" card sends a batch to POST /aa/command-def/import.

1. Import patterns

Accepts XML (any <command>, <cmd>, or <pattern> element with text or name/pattern/value attribute) or CSV (header pattern[,description]).

2. Pool

Pool is empty — import XML/CSV above

3. Bulk Apply — assign profile + category to selected

Pick patterns above, then choose what metadata to apply. Click Apply → batch POST to /aa/command-def/import. Existing (service, ne_profile, pattern) combos are skipped server-side.
